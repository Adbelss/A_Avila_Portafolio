# Sistema de Diseño (UI)

Este documento describe el sistema de diseño y las directrices de UI para `A_Avila_Portafolio`.

## Tokens de diseño

- Colores: paleta principal, secundaria, acentos.
- Tipografía: familia, tamaños, pesos.
- Espaciado: escala base (4px, 8px, 16px...).

## Componentes

- Botones: estilos primario/secundario, estados (hover, focus, disabled).
- Formularios: inputs, labels, validación y mensajes de error.
- Navegación: header, footer, menús.

## Accesibilidad

- Contrastes de color mínimos.
- Navegación por teclado.
